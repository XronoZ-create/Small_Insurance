# Small_Insurance
Микро-бот для автоматического оформления ОСАГО на Альфастрахование, 21 Век, ВСК, Югория и др. страховых компаниях

## Стек

 - Python 3.9
 - Selenium-wire
 - Requests
 - BeautifoulSoup4
 - Pydantic

## Quick Start
```
git clone https://github.com/XronoZ-create/Small_Insurance.git

py -3.9 run.py
```
